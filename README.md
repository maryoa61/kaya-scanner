# ⚡ اسکنر هوشمند آی‌پی کلودفلر و اتوماسیون سابسکریپشن | Kaya Edge Scanner

یک ابزار تحت وب مدرن، کاملاً فرانت‌اند و بدون نیاز به سرور (Serverless) برای تست پینگ واقعی، فیلترینگ و رتبه‌بندی آنی آی‌پی‌های تمیز شبکه **Cloudflare**. این سیستم به طور مستقیم با **Kaya Cloud Panel** و **Cloudflare Worker** همگام‌سازی شده و کانفیگ‌های فرگمنت شما را در لحظه آپدیت می‌کند.

## 🎯 ویژگی‌های برجسته
- **تست پینگ واقعی شبکه داخلی:** ارسال سیگنال مستقیم به اندپوینت `cdn-cgi/trace` برای سنجش دقیق RTT بدون تداخل CORS.
- **توزیع تصادفی الگوریتمی:** انتخاب هوشمندانه آی‌پی از مابین ۱۵ رنج رسمی و مستند شده IPv4 کلودفلر.
- **اتصال آنی به ورکر (Cloud Sync):** ارسال خودکار خروجی مرتب‌شده بر اساس کمترین پینگ به ورکر کلودفلر بدون درگیری با فایل یا کپچا.
- **بدون وابستگی به فیلترشکن:** برای کارکرد دقیق اسکنر، باید فیلترشکن کلاینت قطع باشد تا پینگ واقعی شبکه مخابراتی شما سنجیده شود.

## 🛠️ راهنمای راه‌اندازی سریع روی GitHub Pages
1. یک ریپازیتوری عمومی (Public) به نام `kaya-scanner` بسازید.
2. فایل‌های `index.html` و این راهنما را در ریشه پروژه آپلود کنید.
3. به تب **Settings -> Pages** بروید، شاخه انتشار را روی **main** تنظیم کرده و ذخیره کنید.
4. آدرس پنل اسکنر شما آماده است! کانفیگ مادر و مشخصات ورکر خود را وارد کرده و دکمه اسکن را بزنید.
